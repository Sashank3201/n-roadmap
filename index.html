A 30-Day Field Manual

Network recon
for hackers.

Theory you can already Google. This is the wire. Thirty days of packets, ports, and shells — built so you stop reading and start touching the network.

How each day works

Three blocks. In this order.

📖
Study — read the concept. Watch one video. Absorb the mental model before you touch the keyboard.
⚡
Execute — type every command in a live terminal. Copy-paste teaches nothing.
💬
Ask Claude — paste the prompt into Claude. Get quizzed. Solidify what you just did.

Capture procedure
  1. Open Wireshark, select eth0, start capture.
  2. In another terminal: curl http://example.com.

        |                                    |

        |                                    |
        | ◀── SYN/ACK  seq=0 ack=1  ──────── |   "ok, you talk"
        |                                    |
        | ─── ACK  seq=1 ack=1  ───────────▶ |   "here's data"

practical reference
The chain

  2. ARP — laptop needs gateway MAC: "who has 192.168.1.1?"
  3. TCP SYN to 140.82.114.3:443, via gateway.

  6. Server responds. Page renders. Done.
